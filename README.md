# Task Organizer using C++

## Project Information

* Created On: 7 December 2025
* Project Type: C++ Program
* Category: Task Management System

---

## Overview

This project is a console-based task organizer developed in C++. It helps users manage their daily tasks efficiently by allowing them to add, view, update, and delete tasks.

The project focuses on applying core programming concepts such as data structures, file handling, and modular design.

---

## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Save tasks to file
* Load tasks from file

---

## Technologies Used

* C++
* File Handling (fstream)
* Standard Template Library (STL)

---

## Data Structures

* Vectors or arrays for task storage
* Structures or classes to represent tasks

---

## How to Run

1. Compile the code:
   g++ main.cpp -o task-organizer

2. Run the program:
   ./task-organizer

---

## Project Structure

* main.cpp: main application logic
* tasks.json: task structure and functions
